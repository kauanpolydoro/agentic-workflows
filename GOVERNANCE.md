# Governance

Agentic Workflows uses maintainer-led governance during its initial release phase.
Maintainers review changes, releases, security responses, compatibility claims, and project direction in public wherever security and privacy allow.

Substantial changes should begin with an issue or proposed ADR.
Decisions favor evidence, user safety, portability, maintainability, and a small operational surface.
Consensus is preferred; when it cannot be reached, the active maintainer records the decision and rationale.

Contributor access is earned through sustained, constructive work.
No contributor or sponsor can purchase technical approval, verification status, or moderation outcomes.
