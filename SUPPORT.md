# Support

Use GitHub Discussions for usage questions and workflow-authoring guidance when Discussions are enabled.
Use the issue forms for reproducible bugs, workflow requests, and adapter format problems.

Before filing a bug, run `pnpm awf doctor --json`, remove sensitive paths or values, and search existing reports.
Include the CLI version, operating system, Node version, command, expected behavior, actual behavior, and a minimal safe reproduction.

Security vulnerabilities must follow [SECURITY.md](SECURITY.md), not public support channels.
The project does not provide guaranteed response times before a funded support model exists.
