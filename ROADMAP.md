# Roadmap

## Delivered foundation

- Shipped 20 complete recipes and the strict schema.
- Validated safe install, update, and remove on Linux, macOS, and Windows.
- Published searchable documentation through GitHub Pages.
- Published the CLI and shared core as public npm packages.
- Enabled subsequent package releases through npm trusted publishing with provenance.
- Recorded installation evidence without overstating agent execution.

## Subsequent releases

- Run controlled external-agent verification against the included fixtures.
- Add reviewed evidence files with exact tool versions and environments.
- Improve adapter-specific argument conventions as official formats evolve.
- Expand the catalog through fixture-backed community recipes.

Roadmap items are proposals, not delivery commitments.
