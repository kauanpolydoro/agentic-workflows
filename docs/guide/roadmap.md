# Roadmap

The current launch scope is a validated 20-recipe catalog, a safe local CLI, documented adapters, reproducible fixtures, and an evidence model.

Future releases may add more fixture-backed installation evidence, externally executed agent evidence, package-registry distribution, signed release artifacts, and additional adapters backed by official documentation.

Roadmap items are proposals rather than commitments.
See the repository [ROADMAP.md](https://github.com/kauanpolydoro/agentic-workflows/blob/main/ROADMAP.md) for acceptance signals and sequencing.
