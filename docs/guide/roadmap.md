# Roadmap

The v0.3.0 release candidate contains 21 recipe bundles, schema version 4, a safe local CLI, documented adapters, reproducible fixtures, and the evidence model described in this guide.
The original 20 recipes retain their historical schema version 3 `pass` dispositions, while their schema version 4 metadata variants remain `blocked` until one cross-cutting human migration review is retained.
`resolve-github-issues` remains separately `blocked` until its required human domain review is retained.
The v0.3.0 npm publication remains gated by the recorded cross-cutting and domain reviews.

Future releases may add more fixture-backed installation evidence, evidence from externally executed agents, signed release artifacts, and additional adapters backed by official documentation.

Treat every roadmap item as a proposal, not a promise.
The repository [ROADMAP.md](https://github.com/kauanpolydoro/agentic-workflows/blob/main/ROADMAP.md) lists the acceptance signals and sequencing for each item.
